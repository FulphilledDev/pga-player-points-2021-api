# PGA Player Points 2021 API

API built with node.js and Express including 2021 PGA Player Points data. 

![Imgur](https://i.imgur.com/xKLWGXY.png)

Currently including:

- Player Season ID
- Season
- Player ID
- Name
- World Golf Rank
- World Golf Rank Last Week
- Events
- Average Points
- Total Points
- Points Lost
- Points Gained
